# Totoro
一个基于Angular2+中组件，TypeScript, Bootstrap, TypeScript 和 WebPack的简历模板
#
[demo](http://resume.99diary.com)

### 功能需求

- [x] 基本对象绑定

- [x] 友好的浏览器提示

- [x] 主题切换

- [ ] 简易搜索

- [ ] 多语言支持

- [ ] 导出文档

- [ ] 统计报表

- [ ] 按需加载

- [ ] 打包模板

- [ ] 单元测试
